## System Requirements:
* Python 3 installed
* Administrator privileges

## Installation:
1. Change directory to project folder
`cd ecommerce-store`
2. Activate the virtual environment
`.\Sources\activate`
3. Change directory to source code folder
`cd src`
4. Activate the server by running 
`python manage.py runserver`
