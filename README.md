## System Requirements:
* Python 3 installed
* Administrator privileges

## Installation:
Change directory to project folder
`cd ecommerce-store`
Activate the virtual environment
`.\Sources\activate`
Change directory to source code folder
`cd src`
Activate the server by running 
`python manage.py runserver`
